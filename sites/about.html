What is WaveSearch?

WaveSearch is a modern, high-speed dataset search and lookup engine created for radio and communications enthusiasts. It replaces aging tools like HamDB by offering faster access, cleaner data structures, and a flexible offline mode that works on nearly any system.

Access to Multiple Datasets (Adding more)

WaveSearch doesn’t just stop at amateur radio. It provides unified search access across multiple FCC and FAA datasets, all parsed, filtered, and indexed for instant results:

  • Amateur Radio (HAM): FCC ULS call signs, FRNs, and license data
  • GMRS: General Mobile Radio Service records for family and business use
  • Tower Structures: FCC Antenna Structure Registration (ASR) tower listings
  • Aviation: FAA and FCC registered communications infrastructure data

Why It Matters

The communications landscape depends on accurate, up-to-date data — but existing APIs and services are either slow, incomplete, or commercially restricted. WaveSearch bridges that gap with transparent datasets, open formats, and instant search capabilities without external dependencies or tracking.

Features

  • Instant lookups across multiple dataset types
  • Lightweight, offline-capable JSONL format
  • Rapid local indexing for embedded systems
  • Community-driven and open development model

Built For the Radio Community

WaveSearch was designed to empower operators, developers, and researchers with direct access to public communications data. Whether you’re logging QSOs, mapping tower infrastructure, or analyzing regional networks, it gives you the tools to work without corporate APIs or network restrictions.
